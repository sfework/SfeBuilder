创建数据密钥
示例：key
简写：k

创建本地发布模板
示例：template local [配置文件名]
简写：t l [配置文件名]

创建远程发布模板
示例：template remote [配置文件名]
简写：t r [配置文件名]

使用配置文件发布项目
示例：publish [配置文件名]
简写：p [配置文件名]

生成后文件处理指令：HandleFile
删除文件
delete [文件名]
重命名文件
rename [原文件名] [新文件名]
替换文件内容，注意：只支持简单的字符替换，不支持较长或多行文本
replace [文件名] [预替换内容] [替换内容]

发布前执行命令：BeforeCommands
发布后执行命令：AfterCommands

注意：创建配置模板时[配置文件名]通常为项目名称，不要输入后缀。
	 发布项目时[配置文件名]可以不输入后缀。
